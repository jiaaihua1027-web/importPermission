# 权限导入工具

一个基于 React + Ant Design 的权限导入管理系统前端界面。

## 功能特性

### 1. 权限列表页面
- 权限搜索：按名称、编码搜索
- 多维筛选：按类型、状态筛选
- 增删改查：完整的权限管理功能
- 批量操作：支持批量删除
- 数据导出：导出权限数据
- 分页显示：灵活的分页控制

### 2. 权限导入功能（弹窗方式）
- 模版下载：支持4种模版类型
  - 服务和按钮 + 全部领域
  - 服务 + 全部领域
  - 服务和按钮 + 指定领域
  - 服务 + 指定领域
- 领域选择：树形结构多选领域（财务、供应链、人力、制造、客户、项目等）
- 文件上传：支持 Excel、CSV 格式，拖拽上传
- 导入进度：实时显示导入进度

## 技术栈

- **React 18** - 前端框架
- **React Router 6** - 路由管理
- **Ant Design 5** - UI 组件库
- **Vite** - 构建工具

## 使用流程

### 权限导入流程
1. 访问权限列表页面（默认首页）
2. 点击右上角"权限导入"按钮
3. 选择模版类型（4种类型）
4. 如选择"指定领域"，点击"选择领域"按钮
5. 在领域选择弹窗中勾选需要的领域
6. 点击"下载模版"按钮下载模版文件
7. 填写模版文件
8. 拖拽或点击上传填写好的文件
9. 点击"开始导入"按钮
10. 查看导入进度，等待完成

### 权限管理流程
1. 在权限列表页面查看所有权限
2. 使用搜索框搜索权限
3. 使用筛选器按类型、状态筛选
4. 点击"新增权限"添加新权限
5. 点击编辑图标修改权限
6. 点击删除图标删除权限
7. 勾选多条数据批量删除
8. 点击"导出"按钮导出数据

## 页面导航
